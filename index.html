# Request a Survey Quote System A professional, responsive web form for Benton W. Dewar & Associates that allows clients to request a land survey quote online. ## Features - Contact information form - Property details section - Multi-select survey type options - Project details input - Mobile responsive design - Submission confirmation message ## Survey Types Supported - Boundary Survey - ALTA/NSPS Survey - Construction Staking - Topographic Survey - Subdivision Planning - Not Sure / Need Assistance ## How It Works 1. User fills out form 2. Selects survey type(s) 3. Submits request 4. Confirmation message displays ## Tech Stack - HTML5 - CSS3 - Vanilla JavaScript ## Setup Just open `index.html` in any browser.
